## Features

- **Currency** - A currency system (based on [Dank Memer](https://dankmemer.lol) discord bot)
- **Spawner** - On-the-spot events to join to get some credits.

## Self-Hosting

Though the source code is visible here, please ensure you won't host your public version of this bot for you or with a certain group including you as an interest or for any other stuff. It's only intended for a private server at the moment and will change in the future.
